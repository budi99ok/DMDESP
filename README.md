### Pin on DMD LED P10 Panel

| DMD P10 | NODEMCU | 
| ------- | ------- |
| A       | D0      |
| B       | D6      |
| CLK     | D5      |
| SCK     | D3      |
| R       | D7      |
| NOE     | D8      |
| GND     | GND     |
